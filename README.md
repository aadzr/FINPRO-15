# FINPRO-15
# RSI Strain Index Calculator

## Deskripsi Program
Program berbasis bahasa C untuk menghitung risiko cedera 
tangan dan pergelangan tangan (Repetitive Strain Injury) 
akibat aktivitas gaming atau coding sehari-hari.
Program ini menggunakan metode Strain Index dari 
Moore & Garg (1995) dan mendukung SDG 3: Good Health and Well-Being.
